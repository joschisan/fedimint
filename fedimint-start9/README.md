# Fedimint for Start9

This package allows you to run a Fedimint guardian node on your Start9 server.

## Features

- Run a Fedimint guardian node
- Automatic integration with Bitcoin Core on your Start9 server
- Uses Iroh networking for improved P2P performance
- Web UI for federation management
- API for client connections

## Requirements

- Bitcoin Core running on your Start9 server (required dependency)

## Usage

1. Install the package from the Start9 marketplace
2. The package will automatically connect to your Bitcoin Core instance
3. Access the web UI to set up or join a federation
